About This Project

Smart Energy Consumption Analysis & Prediction System

Project Overview

This Smart Energy Consumption Analysis system is a comprehensive machine learning solution designed to monitor, analyze, and predict household energy consumption at the device level. The system provides actionable insights and personalized recommendations to help reduce energy waste and lower electricity costs.

Technology Stack

Machine Learning

LSTM Neural Networks, Linear Regression

Backend

Python, Flask, TensorFlow, Scikit-learn

Data Analysis

Pandas, NumPy, Matplotlib, Seaborn

Frontend

HTML5, CSS3, JavaScript, Chart.js

Key Features

  • Real-time energy consumption monitoring
  • Device-level energy tracking (Kitchen, Laundry, HVAC)
  • AI-powered consumption prediction using LSTM
  • Interactive dashboards with hourly, daily, and monthly views
  • Smart energy-saving suggestions
  • RESTful API for data access
  • Responsive web interface

Dataset Information

Source: UCI Machine Learning Repository

Dataset: Individual Household Electric Power Consumption

Records: 2,075,259 measurements

Time Period: December 2006 - November 2010 (47 months)

Granularity: Minute-level readings

Features: Global active power, Sub-metering (Kitchen, Laundry, HVAC)

Model Performance

Linear Regression (Baseline)

R² Score: ~85-90%

MAE: ~0.15-0.25 kW

Training Time: Fast

Best Model

LSTM Neural Network

R² Score: ~90-95%

MAE: ~0.10-0.20 kW

Training Time: Moderate

Objectives:

  • Demonstrate end-to-end ML pipeline development
  • Apply LSTM networks to time series forecasting
  • Create practical, user-facing ML applications
  • Develop full-stack web development skills
